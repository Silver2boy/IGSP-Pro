# IGSP-Pro

Institutional Gold Smart Pivot Professional

## Overview

IGSP-Pro is a professional Smart Money Concepts (SMC) trading framework developed specifically for XAUUSD (Gold).

The project combines Institutional Pivot Points, Market Structure, Liquidity Analysis, Order Blocks, Fair Value Gaps, and Risk Management into one professional TradingView indicator.

---

## Main Features

* Institutional Pivot Engine
* Market Structure (BOS / CHOCH)
* Liquidity Detection
* Equal High / Equal Low
* Liquidity Sweep
* Smart Order Blocks
* Fair Value Gap (FVG)
* Inversion FVG
* Premium / Discount Zones
* Trend Engine
* ATR Risk Management
* Dashboard
* Alerts
* Backtesting Strategy
